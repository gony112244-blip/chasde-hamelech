## 10. מבנה ה-API (שרת Backend)

השרת רץ על פורט `3002` בשרת (Nginx מנתב אליו).

### נתיבים ציבוריים
| Method | נתיב | תיאור |
|--------|-------|-------|
| GET | `/api/health` | בדיקת תקינות |
| GET | `/api/stats` | סטטיסטיקות לדף הבית |
| GET | `/api/thank-you` | הודעות תודה מאושרות |
| POST | `/api/thank-you` | שליחת הודעת תודה חדשה |
| POST | `/api/contact` | שליחת פניה |
| POST | `/api/volunteer` | הרשמה להתנדבות |

### נתיבי ניהול (דורשים סיסמה)
| Method | נתיב | תיאור |
|--------|-------|-------|
| POST | `/api/admin/login` | כניסת מנהל |
| GET | `/api/admin/contacts` | כל הפניות |
| GET | `/api/admin/volunteers` | כל המתנדבים |
| GET | `/api/admin/thank-you` | כל הודעות התודה (כולל ממתינות) |
| PUT | `/api/admin/thank-you/:id` | אישור/דחייה הודעה |
| PUT | `/api/admin/stats` | עדכון סטטיסטיקות |

### סיסמת ניהול
נשמרת ב: `/var/www/chasde-hamelech/backend/.env`
```
PORT=3002
ADMIN_PASSWORD=chasde2026
```

## 13. מה חסר / שלבים עתידיים

| פריט | מצב |
|------|-----|
| דומיין `chasde-hamelech.org.il` | נרכש, DNS מוגדר |
| SSL/HTTPS | ממתין להתפשטות DNS — להריץ certbot לאחר שעה-שתיים |

| מייל אוטומטי | לא מחובר עדיין |

### כשה-DNS מוכן — פקודה אחת ל-HTTPS:
