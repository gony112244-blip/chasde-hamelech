## 9. מבנה ה-API (שרת Backend)

השרת רץ על פורט `3002` בשרת (Nginx מנתב אליו).

### נתיבים ציבוריים
| Method | נתיב | תיאור |
|--------|-------|-------|
| GET | `/api/health` | בדיקת תקינות |
| GET | `/api/stats` | סטטיסטיקות לדף הבית |
| GET | `/api/thank-you` | הודעות תודה מאושרות |
| POST | `/api/thank-you` | שליחת הודעת תודה חדשה |
| POST | `/api/contact` | שליחת פניה |
| POST | `/api/volunteer` | הרשמה להתנדבות |

### נתיבי ניהול (דורשים סיסמה)
| Method | נתיב | תיאור |
|--------|-------|-------|
| POST | `/api/admin/login` | כניסת מנהל |
| GET | `/api/admin/contacts` | כל הפניות |
| GET | `/api/admin/volunteers` | כל המתנדבים |
| GET | `/api/admin/thank-you` | כל הודעות התודה (כולל ממתינות) |
| PUT | `/api/admin/thank-you/:id` | אישור/דחייה הודעה |

## 12. מה חסר / שלבים עתידיים

| פריט | מצב |
|------|-----|
| דומיין (chasde-hamelech.co.il) | לא נרכש עדיין |
| SSL/HTTPS | יפעל אוטומטית לאחר דומיין (Certbot) |
| בסיס נתונים (PostgreSQL) | בשלב זה הכל In-Memory (מתאפס ב-restart) |
| מייל אוטומטי | לא מחובר עדיין |

### כשיהיה דומיין — 3 פקודות בלבד:
```bash
ssh root@187.124.168.26

# עדכן Nginx
sed -i 's/server_name _;/server_name YOUR-DOMAIN.co.il www.YOUR-DOMAIN.co.il;/' \
  /etc/nginx/sites-available/chasde-hamelech
nginx -t && systemctl reload nginx

# SSL חינמי
certbot --nginx -d YOUR-DOMAIN.co.il -d www.YOUR-DOMAIN.co.il
```
